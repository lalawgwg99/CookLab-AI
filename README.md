# CookLab AI

料理實驗室 × 台灣食材週期菜單

CookLab AI 用實驗驗證料理成功率，結合台灣當季食材與價格，提供穩定又省錢的每週菜單與實驗報告。

## 開發

1. 安裝依賴
   npm install

2. 本地開發
   npm run dev

3. 產出建置
   npm run build

4. 預覽
   npm run preview

# CookLab AI (English)

Kitchen Lab × Taiwan Seasonal Menu Planner

CookLab AI validates recipes through experiments and combines Taiwan seasonal ingredients with pricing to deliver reliable, budget-friendly weekly menus and test reports.

## Development

1. Install dependencies
   npm install

2. Start dev server
   npm run dev

3. Build
   npm run build

4. Preview
   npm run preview
